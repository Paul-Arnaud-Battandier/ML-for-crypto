🚀 BTC Algorithmic Trading

This project focuses on building an end-to-end Machine Learning pipeline to predict and trade Bitcoin (BTC) on Binance via API.


🎯 Project Goals

Prediction: Develop a high-accuracy ML model to forecast BTC price trends. 

Backtesting: Validate strategies using historical data before live deployment. 

Execution: Automate 24/7 trading using a VPS and the Binance API.


🔬 Research & Methodology

To find the most effective strategy, the project involves a rigorous research phase:

Algorithmic Benchmarking: Comparing several models such as Logistic Regression, Random Forest, SVM, Neural Networks, and XGBoost to find the best fit for crypto volatility. 

Feature Engineering: Selecting and transforming the most relevant market indicators to reduce prediction error. 

Optimization: Fine-tuning hyperparameters for the top-performing models. 

Evaluation: Analyzing performance through metrics like Weighted F1-score and ROC Curves.
